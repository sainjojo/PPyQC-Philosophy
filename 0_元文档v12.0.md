## 0.2 核心概念速查（v12.0 最新版）

### 一、概述

本文件提供 PPyQC 体系核心概念的一句话定义，涵盖哲学核心 v11.4 和通用应用层 A1-A8。

### 二、哲学核心概念

| 概念 | 一句话定义 | 所在章节 |
|------|------------|----------|
| **可能性** | "尚不存在但可以存在"的本体论空间 | 第一章 |
| **不可观测子** | 可能性的无组织态单元，数量无穷、运动无穷快、寿命无穷短，承载关联无限性 | 第一章 |
| **关联无限性** | 任意两个不可观测子之间潜在地存在无限关联，本身是纯粹潜能，不包含任何导向 | 第一章 |
| **导向结构** | 系统维持自身、引导关联的稳定模式。从粒子的守恒律到人类的文化传统，是导向能力从固化到可塑的连续谱 | 第四章、第九章 |
| **元规则** | 可能性走向存在必须支付的入场费：M1载体凝聚与排他、M2涌现随机与导向必然、M3代理必然、M4存续必然 | 第二章 |
| **稀缺性** | 系统因元规则约束而必然面临的资源、能力、协同、存续条件的有限性 | 第三章 |
| **系统** | 从可能性之海中涌现的、具有边界和内部结构的存在。其组织形态即导向结构 | 第四章 |
| **存在活动** | 系统通过核心协同，在具体情境中实现自身延续发展的基本过程单元 | 第六章 |
| **痕迹** | 系统通过其组织形态，对关联无限性中某些关联模式的持续导向能力 | 第六章 |
| **信息** | 具有传播能力的痕迹 | 第七章 |
| **意义场** | 系统在将传播式关联确立为导向层主流后，其导向结构获得的信息化形态 | 第九章 |
| **标记信号** | 传播流中获得稳定导向权重的信息形式 | 第十章 |
| **四核心** | 通用涌现算法在四个维度上的专门化部署：信用（资源）、主动性（行动）、话语权（协调）、意义导向核心（意义） | 第十三章 |
| **通用涌现算法** | 试探→反馈→预期形成→资源配置四步循环，是对M1-M4的系统性应对 | 第四章、第十三章 |
| **意识** | 传播过程中自然形成的、占据主导地位的信号流，发生于功能中层（临时意义场） | 第十五章 |
| **价值** | 导向结构赋予对象的标记，拟合的是对象对系统稀缺性的实际缓解能力 | 第十六章 |
| **存在力** | 系统拓展可能性空间边界的能力 | 第十六章 |
| **分化陷阱** | 导向结构在四个维度上过度专门化后，相互耦合过紧导致的整体适应性丧失 | 第十八章 |
| **努斯** | 导向结构自我反思、自我更新与创造性重构的二階能力 | 第十八章 |
| **量级淘汰** | 巨量失败与少数幸存的普遍现象 | 第十九章 |
| **四维元信念** | 通用涌现算法在四个维度上成功迭代所必须遵循的算法纪律 | 第二十章 |
| **认知1 / 认知2** | 算法在既有标记下的运作 / 算法对自身标记的审视 | 第二十一章 |

### 三、通用应用层概念（A1-A8）

| 概念 | 一句话定义 | 所属模块 |
|------|------------|----------|
| **坐标原点** | 具备相对完整存在性的宏观系统，为分析对象划定了可能性空间的基本边界 | A1 |
| **生态位** | 系统在场域网络中的功能定位——缓解什么稀缺性，依赖什么支持 | A1 |
| **涌现缝隙** | M3代理必然的产物——上级划分的基调与下级遭遇的涌现之间的必然不匹配 | A5 |
| **押注机制** | 话语权博弈的制度化形态——主体以自身话语权为赌注支持方案，根据实际效率接受奖惩 | A5 |
| **元话语权** | 制定和修订博弈规则本身的权力，需要反叙事的独立检验和权力的有限期轮换 | A5 |
| **程序共识** | 各方对博弈程序本身的接受、对结果的接受义务、对修订程序的共识 | A5 |
| **价值共识** | 多系统在信用互动中通过比较机制形成的、对资源价值的集体估计 | A4 |
| **制度底座** | 资源可调用化的制度框架——产权制度、货币制度、经济体身份与账户体系 | A4 |
| **信用分层** | 通用信用、专门信用与自由信用的三层结构，对应不同期限和风险的资源配置需求 | A4 |
| **信息信用** | 导向层储备的内容性标记——关于世界如何运作的知识和模型 | A3 |
| **话语权标记** | 代理了能动性的标记——信任、敌意、归属、关爱等直接驱动注意投向和行为选择 | A3/A5 |
| **事件界面** | 导向结构与现实世界发生直接接触的界面——以主体为中心、以当下为锚点、以行动为目的 | A7 |
| **情境锚定** | 从应激的标记激活到充分的情境部署的完整过程——事件界面的"第一秒" | A7 |
| **意志启动** | 期望性标记的方案博弈产生胜出者，通过扳机标记锁定行为通道的不可逆跃迁 | A7 |
| **策略调度** | 在部署与执行之间划定边界，在机会窗口与程序实现之间选择策略的元能力 | A7 |
| **策略风格** | 导向结构对不确定性的稳定应对模式——长期意义场中积累的元标记在事件界面上的稳定表达 | A7 |
| **构建即重现** | 系统构建不是无中生有的创造，而是对既有导向结构的重新实例化 | A8 |
| **价值摄取** | 一个系统通过与其他系统的互动，获取对自身稀缺缓解能力的贡献 | A8 |
| **意义承载** | 一个系统的存在意义被纳入另一个系统的长期意义场，其存在痕迹被记录，其导向结构被重现 | A8 |
| **存续悬置** | 价值摄取与意义承载失衡的病理形态——系统保留了部分主体性，却无法在宏观系统中获得存在意义的托付 | A8 |
| **干涉接口** | 干涉者与系统之间进行导向性信息传递的通道，内在于系统生态的既有交换点 | A8 |

### 四、系统核心口诀

- **信用**：创造时空自由——算法在资源维度的部署

### 二、应用层核心线索

从A1到A8，通用应用层模块遵循一条清晰的逻辑线索：

**从诊断到实践再到干预。**

A1提供通用的系统诊断方法论。A2至A5分别锚定系统在操作层、导向层、信用侧和话语权侧的实现条件——这些是"健康系统需要什么"。A6将这套方法论应用于具体实例，验证其适用性。A7收束于事件界面——导向结构如何在具体情境中被实时激活、被实践校准、被行动兑现——回答"在给定条件下如何做出更好的选择"。A8最终转向系统间的构建关系与干涉的可能——回答"系统为何是现在这个样子，以及如何在病理中恢复自我校准的能力"。

### 三、入门路径

**路径A：从导向结构出发（推荐理论研究者）**
1. 阅读哲学核心第一章至第四章，理解可能性元根基与导向结构的涌现。
2. 阅读第九章和第十二章，理解意义场与导向结构的分化。
3. 依次阅读第十三章至第二十二章，把握四核心、分化陷阱、努斯与四维元信念。

**路径B：从应用模块入手（推荐实践者）**
1. 阅读A1系统分析学总览，建立诊断的方法论框架。
2. 根据关注领域选择性阅读A2-A5中的相关模块。
3. 阅读A7理解事件界面上的行动自由。
4. 阅读A8理解系统间的构建关系与干涉的可能。
5. 遇到概念不清晰时，回溯到哲学核心对应章节。

**路径C：从价值问题出发（推荐哲学背景）**
1. 阅读哲学核心第三章和第十六章，理解稀缺性与价值作为导向标记。
2. 阅读第二十章，理解四维元信念作为算法纪律。
3. 阅读A4和A5，理解价值共识和话语权协调的制度化实现。
