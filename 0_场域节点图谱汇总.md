# 第一部分：场域节点图谱总集

## 一、场域图谱的核心逻辑

场域节点图谱的核心逻辑可以概括为：

> **情感场域提供动力环境 → 驱动主动性实践（传播基础 + 主体-事件） → 产出知识（可能性环境）和文化（选择环境） → 二者反馈回动力和实践，形成完整闭环。**

### 1.1 三大环境与一个实践

| 组件 | 对应场域 | 核心功能 | 回答的问题 |
|------|----------|----------|------------|
| **动力环境** | F01 情感场域 | 提供好奇、恐惧、渴望、无聊等情绪动力 | “为什么要动？” |
| **主动性实践** | F02 传播基础 + F07 主体-事件 | 从无定向到定向，追问、操作、执行 | “如何行动？” |
| **可能性环境** | F03 认知、F04 理解、F06 信息媒介 | 产出知识：概念、模型、理论、信念、知识体系 | “什么可能发生？” |
| **选择环境** | F05 社会传播、F09 话语权 | 产出文化：共识、规范、价值观、规则 | “什么值得选？” |

### 1.2 闭环图景

```
┌─────────────────────────────────────────────────────────────────┐
│                       动力环境                                  │
│                    （F01 情感场域）                             │
│              好奇、恐惧、渴望、无聊...                          │
└─────────────────────────┬───────────────────────────────────────┘
                          │ 提供动力
                          ↓
┌─────────────────────────────────────────────────────────────────┐
│                    主动性实践                                   │
│        （F02 传播基础场域 + F07 主体-事件场域）                 │
│          定向、追问、操作、执行                                 │
└─────────────┬───────────────────────────┬───────────────────────┘
              │                           │
              ↓                           ↓
┌─────────────────────────┐   ┌─────────────────────────────────┐
│     可能性环境          │   │        选择环境                  │
│   （知识侧）            │   │      （文化侧）                  │
│ F03认知：命名、分类     │   │ F05社会传播：共识、文化规范      │
│ F04理解：概念、模型、   │   │ F09话语权：规则、价值观、习俗    │
│   理论、信念、知识体系  │   │                                 │
│ F06信息媒介：信息形式   │   │                                 │
│ “什么可能发生”         │   │ “什么值得选择”                  │
└─────────────────────────┘   └─────────────────────────────────┘
              ↑                           ↑
              └─────────── 反馈 ───────────┘
        （知识更新文化，文化引导知识探索）
```

### 1.3 核心结论

- **动力环境**驱动**主动性实践**，使系统从无定向状态进入具体场域操作。
- **主动性实践**在具体场域中产出**知识（可能性环境）** 和**文化（选择环境）**。
- **知识**回答“世界如何运作”，**文化**回答“什么值得追求”。
- 二者相互反馈：新知识可能改变文化规范，文化规范引导知识探索的方向。
- 整个闭环持续演化，推动系统存在力的提升。

# 第二部分：场域节点图谱使用指南

## 一、定位

场域节点图谱是**2.3符号体系的一个专业优化工具**，是对信息信用的结构化明确。它将PPyQC哲学核心中关于信息侧存在活动的抽象概念，具象化为14个相互关联的场域和可操作的节点簇。

**本质**：对符号体系工具本身的优化，降低认知负荷，提供统一的知识线索。

**与PPyQC体系的关系**：
- 通用基础工具（2.1-2.12）提供分析框架
- 场域节点图谱提供框架内的**领域知识节点**
- 使用时按需加载（如同2.3的插件机制）

---

## 二、使用场景

### 场景1：思维范式参考

**需求**：快速理解PPyQC体系中“信息侧存在活动”的全貌。

**做法**：阅读《场域节点图谱总集》的“核心逻辑”章节，理解动力环境→实践→可能性环境+选择环境的闭环；浏览14个场域的定位和速查表。

**产出**：对信息侧存在活动的结构化认知。

---

### 场景2：系统分析与诊断

**需求**：分析一个组织的创新停滞问题。

**做法**：
1. 定位相关场域：F02传播基础（定向动力）、F04理解（深度加工）、F07主体-事件（执行）
2. 检查节点激活度：好奇、疑问、追问How是否活跃？对比、抽象是否缺失？
3. 追踪调用链：F02是否主动调用F10资源信用（追问方法）？
4. 输出诊断：定位为“定向动力缺失”或“深度加工不足”

**产出**：诊断报告 + 干预建议（可对接2.8病理学）。

---

### 场景3：创作支持

**需求**：设计一个“人物觉醒”情节。

**做法**：
1. 从F02传播基础出发：无聊→疑问（“我为什么活着？”）→追问What/Why/How
2. 调用F01情感：焦虑、渴望
3. 进入F04理解：对比不同价值观、形成新信念
4. 进入F07主体-事件：制定计划、采取行动

**产出**：情节节点序列 + 可直接使用的描述词汇（如wonder, clarify, insight, resolve）。

---

### 场景4：理论研究

**需求**：验证“信息媒介场域”的节点完备性。

**做法**：
1. 列出信息操作全生命周期：产生→编码→传输→存储→检索→解码→使用→销毁
2. 对照F06信息媒介场域的各层节点
3. 发现缺失（如“压缩”）→ 建议补充

**产出**：完备性报告 + 节点扩展建议。

# 第三部分：各场域详细节点图谱

## F01 情感场域（传播动力源）

### 场域定位

情感机制场域是传播子系统的基础动力源。它通过正负反馈情绪改变意义场中信息簇的情感权重，驱动价值评价、信念形成和态度沉淀，从而为所有信息处理活动提供持续动力。

**核心机制**：
- **正反馈**：强化当前态度的权重
- **负反馈**：强化相对态度的权重（不是削弱当前态度）
- **削弱**：仅在替代（新态度取代旧态度）或遗忘（痕迹自然衰减）时发生
- **对立信念共存**：任何信念都存在其对立信念，这是认知的基本结构

---

## F02 传播基础场域（完整版 v2.1）

### 场域定位

**上层信息层·传播基础场域**是从**无定向传播动力**到**定向情境期望**的转化器，是存在活动最初始发生的场域。它处理无聊、随机联想等无定向状态，通过追问What、Why、How、Who/权威，主动调用信息媒介、情感、资源信用、话语权等场域，形成定向期望，从而驱动系统进入具体的存在活动。

**核心任务**：将无定向的传播动力转化为定向的情境期望，并在此过程中明确情境内容、追问意义、询问方法、协调权威。

**核心机制**：
- **追问What** → 调用信息媒介场域（明确情境内容）
- **追问Why** → 调用情感场域/理解场域（追问意义、价值）
- **追问How** → 调用资源信用场域（询问方法、能力、资源）
- **追问权威** → 调用话语权场域（协调规则、权威）

## F04 理解场域（完整版 v2.1）

### 场域定位

**上层信息层·理解场域**是信息处理系统的**深度加工车间**，负责对已有信息信用进行再加工，将“已知/命名”转化为“理解/洞察”，形成概念、模型、理论、体系、信念、知识体系等结构化知识。它是信息到意义的跃迁枢纽。

**核心任务**：深度加工信息，建构概念、模型、理论、信念，实现从信息到知识再到智慧的跃迁。

**核心操作**：对比（Compare）是理解场域的基本原子操作——通过对比，识别异同、抽象属性、分析结构、综合整体、演绎推理。

## F07 主体-事件场域（完整版 v2.1）

### 场域定位

**中层连接层·主体-事件场域**描述**事件的发生与变化**——事件本质是物的变化、关联的实现、运动。本场域刻画主体如何通过操作、工具、机械、动力将意图转化为实际变化，涵盖事件的认知（因果、时间）、事件的发生条件（资源、机会、自由度）、事件的操作流程（工程阶段），以及事件的主观评价（价值判断、命运/自由）。它是上层意图与下层物理世界之间的执行层。

**核心任务**：将意图转化为行动，驱动事件发生，实现从信息到物质-能量变化的跃迁。

## F10 资源信用场域（完整版 v2.2）

### 场域定位

**中层连接层·资源信用场域**描述资源的**生产、获取、持有、交易、估值**，以及支撑这些活动的**货币、账户、市场、财政**体系。它是资源流动与价值交换的核心，向上对接情感场域（贪婪、渴求）和理解场域（价值判断），向下对接具体场域的生产活动和资源载体。

**核心任务**：管理资源的配置与交换，支撑价值计量与信用积累。
