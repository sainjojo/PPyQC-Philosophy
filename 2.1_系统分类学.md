## 2.1.0 系统分类学总览

### 一、模块定位

系统分类学是 PPyQC 体系中研究**存在如何被识别、描述和分类**的通用基础工具。在哲学核心中，“存在即系统”——一切存在都以系统形态呈现，其组织形态即**导向结构**。分类学的任务正是刻画不同存在的导向结构在各个维度上的静态特征，为理解和分析存在提供多维框架。

本模块与 **2.2 系统动力学** 相互配合，共同构成理解存在的完整视角：
- **分类学**提供系统的“解剖学”——系统在当前约束下，其导向结构呈现出怎样的配置？能够打开怎样的可能性空间？
- **动力学**提供系统的“生理学”——导向结构如何随时间演化？可能性边界如何移动？

**核心线索**：分类学的四个层次（L1-L4）是对**导向结构**在不同侧面上的静态快照——从物质基底到有效覆盖范围，从内部配置到历史相位。

| 层次 | 名称 | 核心问题 | 与导向结构的关系 | 包含维度 |
| :--- | :--- | :--- | :--- | :--- |

| **L2** | 台阶与边界层 | 系统处于什么技术-认知台阶？其可能性空间的客观边界在哪里？ | 刻画导向结构的**有效覆盖范围**——可能性空间就是导向结构的当前投影。 | 生产力台阶、认知框架台阶、存在层级、空间边界、时间期限 |

## 2.1.1 基底约束层：载体、资源、稀缺性与成本

### 一、概述

基底约束层是系统分类学的第一层，回答一个根本问题：**系统由什么构成？面临什么硬约束？**

在 PPyQC 的视角下，基底约束层刻画的是系统**导向结构的物质基底**——载体类型、稀缺性结构、成本特征共同决定了导向能力的硬边界。它是所有上层结构（台阶、动力、演化）的物质前提：没有载体就没有存在，没有资源就没有活动，没有成本就没有选择。

**本层包含四个维度**：
1. **载体类型**：系统由什么构成？（物质/能量/信息/混合）
2. **组织形态复杂度**：系统内部如何组织？（简单/复杂/层级）
3. **稀缺性结构**：系统面临什么样的资源约束？（四类稀缺度）
4. **成本特征**：系统运行消耗什么？成本结构如何？

### 二、与哲学核心的对接（v11.4）

| 哲学核心概念 | 在本层中的体现 |
| :--- | :--- |
| **可能性元根基** | 载体是可能性实现为存在时所凝聚的“锚点” |
| **元规则 M1（载体凝聚与排他）** | 载体类型和组织形态复杂度直接体现 M1；资源稀缺的根源 |
| **元规则 M3（代理必然）** | 能力稀缺的根源 |
| **元规则 M4（存续必然）** | 时间稀缺和存续稀缺的根源；成本特征中的时间成本直接体现 M4 |
| **导向结构** | 本层刻画导向结构的**物质基底**。载体不仅是物质的“容器”，更是导向结构的“锚点”——系统的组织形态就是其最基础的导向结构 |
| **意义场的连续谱** | 自然层存在（如粒子）的原始意义场，其导向结构高度固化，直接由其载体和物理定律决定 |
| **三层模型** | 不同载体类型大致对应不同层次：物质/能量→下层，信息→上层，混合→跨层 |

### 三、维度一：载体类型

载体是存在的基础。没有载体，组织形态无处依附。根据载体的性质，存在可分为三大类及混合型。

#### 3.1 物质存在

| 属性 | 说明 |
| :--- | :--- |
| **定义** | 以物质为主要载体的存在。物质载体具有质量、占据空间、可被感官直接或间接感知。 |
| **载体特征** | 原子、分子、基本粒子等，具有静止质量，服从物理定律。 |
| **组织形态形式** | 空间排列、结构、形状、晶体结构等。 |
| **成本特征** | 运输成本（与距离正相关）、存储成本（与体积/质量正相关）、加工成本（需能量输入）、时间成本（自然衰减）。 |
| **导向结构特征** | 物质存在的导向结构高度固化。例如，一块岩石的“导向”就是其物理化学性质——它在特定条件下只能以特定方式响应。这是最原始的“意义场”。 |
| **示例** | 岩石、天体、人体器官、机器设备、建筑物。 |
| **系统性强度** | 从简单（基本粒子）到复杂（地质结构）跨度大，但通常缺乏功能整合。 |

#### 3.2 能量存在

| 属性 | 说明 |
| :--- | :--- |
| **定义** | 以能量为主要载体的存在。能量载体不具有静止质量，但具有做功能力。 |
| **载体特征** | 能量量子、场、辐射等，服从能量守恒与转换定律。 |
| **组织形态形式** | 分布模式、转换关系、能级分布、能量流等。 |
| **成本特征** | 转换损耗（热力学第二定律）、传输损耗（电阻、辐射）、存储衰减（自放电）、采集成本。 |
| **导向结构特征** | 能量存在的导向结构表现为能量流动的“路径依赖”——水流向低处、电流向低电位。这种导向比物质存在更动态，但仍受物理定律严格约束。 |
| **示例** | 电磁场、热力学系统、核反应堆、声波、光波。 |
| **系统性强度** | 通常具有动态结构，可形成稳定模式（如驻波），但边界往往模糊。 |

#### 3.3 信息存在

| 属性 | 说明 |
| :--- | :--- |
| **定义** | 以信息为主要载体的存在。信息载体承载的是组织形态本身，而非物质或能量。 |
| **载体特征** | 符号、信号、神经脉冲、DNA、文字、数字编码等。信息必须依附于某种物理载体，但信息的“意义”可相对独立。 |
| **组织形态形式** | 语法结构、语义网络、逻辑关系、文化模式等。 |
| **成本特征** | 采集成本（传感器、数据入口）、处理成本（计算资源）、存储成本（介质寿命）、传播成本（带宽）、认知成本（注意力）。 |
| **导向结构特征** | 信息存在的导向结构最为复杂和可塑。从 DNA 的遗传信息（固化）到人类的文化传统（可塑），信息存在的意义场呈现出从原始到反思的**连续谱**。 |
| **示例** | 语言、文化、软件、信念体系、数学定理、音乐旋律。 |
| **系统性强度** | 可高度整合（如文化体系），也可极为简单（如一个符号）。信息存在的系统性往往体现在其内部逻辑关系上。 |

#### 3.4 混合型存在

大多数复杂存在是多种载体类型的混合：

| 类型 | 说明 | 示例 |
| :--- | :--- | :--- |
| **物质-能量混合** | 物质载体携带能量 | 运动的物体、高温物质 |
| **物质-信息混合** | 物质载体承载信息 | 书籍（纸张+文字）、DNA（分子+遗传信息） |
| **能量-信息混合** | 能量载体传递信息 | 电磁波通信（光波+调制信号） |
| **物质-能量-信息混合** | 三者共存 | 人脑（物质+生物电+意识）、计算机（硬件+电能+软件） |

**分类原则**：以主导载体为准。例如，一本书虽然由纸张构成，但其本质功能是传递信息，因此归为信息存在。

### 四、维度二：组织形态复杂度

组织形态是载体上呈现的结构。根据内部组织的复杂程度，可分为三个层次。

#### 4.1 简单存在

| 属性 | 说明 |
| :--- | :--- |
| **定义** | 组织形态单一，无明显内部层次分化的存在。 |
| **特征** | 结构简单，各部分功能未分化，整体行为可由局部属性直接解释。 |
| **示例** | 基本粒子、简单分子、单一符号、基本能量量子。 |
| **系统性体现** | 具有边界和内部结构，但结构极为简单，缺乏子系统分化。 |
| **导向结构** | 完全由物理/化学定律决定，无任何“选择”空间。这是意义场连续谱的起点（原始意义场）。 |

#### 4.2 复杂存在

| 属性 | 说明 |
| :--- | :--- |
| **定义** | 内部有分化，但无严格层级嵌套的存在。 |
| **特征** | 不同部分承担不同功能，但各部分之间是网络状关系而非树状层级；整体行为不能简单还原为局部属性。 |
| **示例** | 有机分子、细胞器、小型生态系统、简单社会组织、复杂符号系统。 |
| **系统性体现** | 功能分化出现，但尚未形成稳定的多层嵌套结构。 |
| **导向结构** | 出现初步的“统计倾向”——某些关联模式更易实现。这是从原始意义场向学习意义场过渡的区域。 |

#### 4.3 层级存在

| 属性 | 说明 |
| :--- | :--- |
| **定义** | 内部有多层嵌套结构，每一层都是下层的整合形态。 |
| **特征** | 形成清晰的“部分-整体”层级，上层对下层具有约束和控制，下层为上层提供基础和功能；每一层都有相对自主性，但又受上层整合。 |
| **示例** | 多细胞生物（细胞→器官→个体）、大型社会组织（个人→部门→组织→社会）、复杂软件系统（模块→组件→系统）。 |
| **系统性体现** | 具有完整的层级结构，每一层都涌现出新的性质和功能，是系统性最强的形态。 |
| **导向结构** | 出现多层导向：下层固化（生理本能），上层可塑（学习、文化）。这是意义场连续谱的中高段。 |

#### 4.4 复杂度与系统性强度

| 复杂度 | 系统性强度 | 典型特征 | 意义场状态 |
| :--- | :--- | :--- | :--- |
| **简单** | 弱 | 结构单一，无功能分化 | 原始意义场（完全固化） |
| **复杂** | 中 | 功能分化，网络状关系 | 学习意义场萌芽 |
| **层级** | 强 | 多层嵌套，涌现性质 | 可具备反思意义场（高阶） |

### 五、维度三：稀缺性结构

稀缺性是元规则在系统层面的动态投影。本维度刻画系统面临的四类稀缺性的强度。

| 子维度 | 定义 | 对应元规则 | 量化参考（高/中/低） |
| :--- | :--- | :--- | :--- |
| **资源稀缺度** | 系统所需物质、能量、信息载体的短缺程度 | M1 载体凝聚与排他 | 资源缺口/总需求、价格波动率 |
| **能力稀缺度** | 系统内部专业能力的匮乏程度 | M2 涌现随机与导向必然 + M3 代理必然 | 关键岗位空缺率、培训成本占比 |
| **协同稀缺度** | 多主体协调能力的有限性 | M3 代理必然 | 协调成本/总成本、共识达成周期 |
| **存续稀缺度** | 系统维持自身存在的条件的有限性 | M4 存续必然 | 时间窗口紧迫度、传承断裂风险 |

**稀缺性与基底约束的关系**：
- 载体类型影响资源稀缺：物质存在受资源稀缺约束最强，信息存在最弱。
- 组织形态复杂度影响能力稀缺：层级存在需要更多协调能力，能力稀缺度通常更高。
- 成本特征与稀缺性互为表里：高稀缺性意味着高成本。

### 六、维度四：成本特征

成本是系统为克服稀缺性而支付的代价。本维度描述系统运行的经济性。

| 子维度 | 定义 | 示例 |
| :--- | :--- | :--- |
| **主导成本类型** | 系统消耗的主要资源类型 | 资源成本、信息成本、能力成本、时间成本、混合 |
| **成本结构类型** | 固定成本与可变成本的比例 | 固定成本主导、可变成本主导、混合 |
| **成本敏感性** | 系统对成本变化的敏感程度 | 高/中/低 |

**成本特征与载体类型的关联**：

| 载体类型 | 主导成本 | 典型成本结构 |
| :--- | :--- | :--- |
| 物质 | 运输/存储/加工成本 | 固定成本较高 |
| 能量 | 转换/传输/存储成本 | 可变成本较高 |
| 信息 | 采集/处理/传播成本 | 固定成本高（首次生产），可变成本低（复制） |

### 七、基底约束层与上层的关系

| 上层维度 | 本层提供的输入 | 作用 |
| :--- | :--- | :--- |
| **生产力台阶** | 载体类型、资源稀缺度 | 决定系统能调用什么资源来发展生产力 |
| **认知框架台阶** | 能力稀缺度、信息成本 | 能力稀缺度越高，认知框架越难以跃迁 |
| **动力结构** | 组织形态复杂度、协同稀缺度 | 复杂度越高，需要的协调能力越强，功能分化越精细 |
| **演化位置** | 成本特征、存续稀缺度 | 高存续稀缺度可能迫使系统进入防御模态或加速演化 |


## 2.1.2 台阶与边界层：技术-认知台阶与可能性边界

### 一、概述

台阶与边界层是系统分类学的第二层，回答一个核心问题：**系统处于什么技术-认知台阶？其可能性空间的客观边界在哪里？**

在 PPyQC 的视角下，系统的可能性空间不是无限的，而是由其物质-能量转化能力（生产力）和信息组织能力（认知框架）共同划定的。本层刻画导向结构的**有效覆盖范围**——生产力台阶和认知框架台阶划定了系统在当前导向结构下能够触及的可能性空间边界。

**本层包含五个维度**：
1. **生产力台阶**：系统转化物质-能量的能力处于什么水平？
2. **认知框架台阶**：系统组织和解释信息的理性工具处于什么水平？
3. **存在层级**：系统在什么时间尺度上存在？
4. **空间边界**：系统占据多大空间？如何与环境交换？
5. **时间期限**：系统持续多久？如何跨越代际？

**核心洞见**：
- 生产力台阶决定系统的**客观可行性边界**——在给定技术下，什么“可能”变成“能”。
- 认知框架台阶决定系统的**主观可欲性边界**——系统“能看到”什么可能性，认为什么值得追求。
- 两者共同构成导向结构在当前条件下能够投射的可能性空间范围。

### 二、与哲学核心的对接（v11.4）

| 哲学核心概念 | 在本层中的体现 |
| :--- | :--- |
| **可能性空间** | 本层直接刻画可能性空间的客观边界——由技术-认知台阶和时空边界共同划定。 |
| **导向结构** | 本层刻画导向结构的**有效覆盖范围**——系统在当前导向结构下能够触及和实现的可能性集合。 |
| **元规则 M1（载体凝聚与排他）** | 生产力台阶受限于 L1 的载体和资源约束；空间边界是 M1 的直接投影。 |
| **元规则 M4（存续必然）** | 时间期限是 M4 的直接投影；认知框架台阶的跃迁是不可逆的相变。 |
| **意义场的连续谱** | 存在层级体现了意义场从原始到反思的连续展开：自然层（原始）→生命层→社会层→个体层（反思）。 |
| **三层模型** | 生产力台阶主要对应下层生产层和中层连接层；认知框架台阶主要对应上层信息层。 |
| **认知1与认知2** | 认知框架台阶的跃迁，本质上是认知2的觉醒——系统开始反思自身的认知框架。 |

### 三、维度一：生产力台阶

生产力台阶刻画系统**转化物质-能量、传输信息**的硬能力。它是系统可能性空间的**物质基础**。

#### 3.1 生产力台阶的层级

| 台阶 | 能源技术 | 材料与制造 | 信道技术 | 典型系统 |
| :--- | :--- | :--- | :--- | :--- |
| **采集狩猎** | 人力、火 | 石器、木器 | 口传、信号 | 早期人类、部落 |
| **农业** | 畜力、水力、风力 | 金属冶炼、陶器、纺织 | 文字、驿站 | 古代文明、帝国 |
| **工业** | 蒸汽、电力、化石燃料 | 钢铁、化工、机械制造 | 印刷、电报、电话 | 近代国家、企业 |
| **信息** | 核能、可再生能源 | 半导体、复合材料、精密制造 | 互联网、移动通信、卫星 | 现代社会、跨国企业 |
| **智能** | 可控核聚变（探索中） | 纳米材料、生物制造 | AI、量子通信、脑机接口 | 前沿科技组织、未来文明 |

**信道技术**的专门说明：信道能力（采集、传输、处理、存储信息的速度和容量）本质上是生产力的一部分——光纤、芯片、存储介质都是物质-能量技术的产物。它受限于材料科学和能源技术，而非认知框架。

#### 3.2 生产力台阶的评估维度

| 子维度 | 核心问题 | 评估指标示例 |
| :--- | :--- | :--- |
| **能源水平** | 系统能调用什么等级的能量？ | 人均能耗、能源结构、能量转化效率 |
| **材料与制造** | 系统能加工什么等级的材料？ | 材料强度、精度、复杂度、生产规模 |
| **信道水平** | 系统能多快、多准地传输和处理信息？ | 带宽、延迟、存储容量、计算能力 |

### 四、维度二：认知框架台阶

认知框架台阶刻画系统**组织和解释信息**的理性工具水平。它是系统可能性空间的**认知边界**——决定了系统“能看到什么可能”。

#### 4.1 认知框架台阶的层级

| 台阶 | 主导理性工具 | 核心特征 | 能“看到”什么 | 典型系统 |
| :--- | :--- | :--- | :--- | :--- |
| **原始思维** | 万物有灵、神话叙事、巫术因果 | 主客不分，以拟人化解释世界 | 直接感知的对象；超自然力量解释一切 | 早期人类部落、儿童早期认知 |
| **经验思维** | 常识推理、技艺传承、经验归纳 | 基于重复经验形成规则，但不追问原理 | 可重复的经验模式；简单的因果链条 | 传统农业社会、手工艺行会 |
| **逻辑思维** | 形式逻辑、演绎推理、抽象概念 | 从公理或定义出发推导普遍必然的真理 | 数学和哲学的对象；超越经验的普遍性 | 古希腊哲学、欧几里得几何 |
| **实证-实验思维** | 控制变量、假设检验、数学化 | 通过可重复的实验验证或证伪假设 | 可量化、可重复的自然规律 | 现代科学共同体、工程研发实验室 |
| **系统思维** | 整体性、涌现性、反馈、演化 | 关注要素间的相互作用和整体行为 | 复杂系统的结构、行为和相变 | 生态学、控制论、复杂性科学 |
| **反思性系统思维** | 系统思维 + 认知2（对认知框架本身的审视） | 能够反思和重构自身的认知框架 | 可能性空间本身、认知的边界、价值指向的演化 | PPyQC 体系、元认知训练、二阶控制论 |

**注意**：这些台阶是**层积**的，而非替代的。高台阶系统仍保留低台阶的能力，但能用高台阶工具整合和超越低台阶。

#### 4.2 认知框架台阶的评估维度

| 子维度 | 核心问题 | 评估指标示例 |
| :--- | :--- | :--- |
| **主导理性工具** | 系统最信任什么方式获取真理？ | 启示/权威/经验/理性/直觉/实证的权重 |
| **认知2自觉度** | 系统在多大程度上反思自身的认知框架？ | 元认知实践频率、对自身偏见的觉察程度 |
| **意义场弹性** | 系统的意义场枢纽是否允许更新？ | 反信念容纳度、经典可再解释性 |

#### 4.3 生产力台阶与认知框架台阶的关系

| 关系类型 | 说明 | 历史例证 |
| :--- | :--- | :--- |
| **生产力支撑认知** | 没有文字（生产力），逻辑思维难以大规模展开；没有印刷术，科学革命难以扩散。 | 古登堡印刷术与宗教改革、科学革命 |
| **认知引导生产力** | 科学思维（认知框架）指导了蒸汽机（生产力）的发明；系统思维指导了生态技术和 AI 的发展。 | 热力学与蒸汽机、信息论与通信技术 |
| **不同步的张力** | 生产力台阶和认知框架台阶可能不同步。 | 中世纪中国：高生产力（农业、冶金），但认知框架停滞于经验-权威型 |

#### 4.4 生产力-认知交叉矩阵

| 组合类型 | 生产力台阶 | 认知框架台阶 | 特征 | 历史例证 | 风险 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **同步领先型** | 高 | 高 | 物质能力与认知能力协同发展，可能性空间全面打开 | 科学革命时期的西欧、当代前沿科技 | 可能过度乐观，忽视系统复杂性 |
| **高生产力-低认知型** | 高 | 低 | 用先进技术服务于僵化或原始的价值框架 | 中世纪中国、用 AI 强化监控的威权社会 | 技术滥用、意义迷失、压制创新 |
| **低生产力-高认知型** | 低 | 高 | 有先进的价值理想和认知框架，但缺乏实现手段 | 早期社会主义运动、古希腊的蒸汽玩具 | 空想停滞、理想无法落地 |
| **同步滞后型** | 低 | 低 | 物质与认知均处于低位，可能性空间狭小 | 与世隔绝的原始部落、崩溃后的文明残余 | 被淘汰、被征服 |

### 五、维度三：存在层级

存在层级刻画系统在**时间尺度**上的定位——由记录载体和更新周期决定。在 v11.4 中，存在层级直接体现**意义场的连续谱**。

| 层级 | 记录载体 | 更新周期 | 主导载体 | 意义场状态 | 系统性强度 | 示例 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **自然层** | 物理常数、地质结构 | 亿年级 | 物质 | 原始意义场：导向结构完全由物理定律固化 | 弱 | 山脉、河流、原子、恒星 |
| **生命层** | DNA | 百万年级 | 物质+信息 | 原始→学习意义场：从基因固化到神经可塑 | 强 | 细胞、植物、动物、生态系统 |
| **社会层** | 文化、制度、语言 | 百年至千年级 | 信息为主 | 学习→反思意义场：文化传统可塑；轴心文明出现反思萌芽 | 强 | 家庭、部落、国家、文明 |
| **技术层** | 工具、知识库、技术文档 | 几年至几十年 | 信息+物质 | 学习意义场：技术知识高度可塑 | 中到强 | 石器、机器、计算机、AI系统 |
| **个体层** | 记忆、信念、个人知识 | 十几年 | 信息为主 | 反思意义场：自我意识、元认知 | 强 | 人的心智、人格、自我认同 |
| **行为层** | 临时决策、动作、事件记录 | 瞬间到几天 | 信息+能量 | 情境依赖：临时性意义场 | 弱 | 一次决策、一次对话、一次交易 |

**跨层关系法则**：
- **慢层为快层提供基础约束**：自然层约束所有上层存在；生命层约束个体层和社会层；社会层约束技术层和行为层。
- **快层为慢层提供创新试错**：行为层的试错累积影响个体信念；个体创新通过社会传播影响文化；技术突破可能改变生命层演化。

### 六、维度四：空间边界

空间边界刻画系统在**空间**上的存在形式——由存在范围、交换界面、边界管控三要素构成。

#### 6.1 存在范围

| 类型 | 定义 | 特征 | 成本特征 | 示例 |
| :--- | :--- | :--- | :--- | :--- |
| **局域存在** | 存在范围局限于有限空间 | 边界明确，内外区分明显 | 运输成本低 | 一块岩石、一栋建筑、一个细胞 |
| **广域存在** | 存在范围覆盖较大空间 | 边界可识别，内部有密度梯度 | 运输成本随距离增加 | 一片森林、一座城市、一个国家 |
| **全球存在** | 存在范围覆盖全球或近乎全球 | 边界模糊，内部高度连通 | 运输成本极高，但可网络化降本 | 互联网、全球贸易网络、世界性宗教 |

#### 6.2 交换界面

| 类型 | 定义 | 特征 | 成本特征 | 示例 |
| :--- | :--- | :--- | :--- | :--- |
| **开放存在** | 交换界面高度通透 | 输入输出自由，变化快 | 交易成本低，外部风险高 | 活细胞、市场经济体、在线社交平台 |
| **封闭存在** | 交换界面极少 | 自给自足，变化慢 | 防御成本低，信息成本高 | 休眠种子、闭关国家、加密系统 |
| **半开放存在** | 交换界面选择性开放 | 平衡开放与安全 | 需支付选择性管控成本 | 多数生物体、防火墙保护的企业 |

#### 6.3 边界管控

| 类型 | 定义 | 特征 | 成本特征 | 示例 |
| :--- | :--- | :--- | :--- | :--- |
| **刚性边界** | 边界明确且难以改变 | 对外防御高，内部秩序严格 | 防御成本高，灵活性差 | 军事要塞、版权保护内容、强隔离系统 |
| **柔性边界** | 边界可调整，管控灵活 | 根据环境变化调整 | 调整成本适中 | 变形虫、弹性企业、自适应软件 |
| **模糊边界** | 边界不清晰，管控松散 | 内外交融，身份模糊 | 管控成本低，可能资源耗散 | 文化影响区、云服务、虚拟社区 |

### 七、维度五：时间期限

时间期限刻画系统在**时间**上的存在形式——由期限长度、期限来源、传承能力构成。

#### 7.1 期限长度

| 类型 | 定义 | 特征 | 存续稀缺度 | 时间成本特征 | 示例 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **短期型** | 期限短，变化快 | 波动大，适应性强，稳定性弱 | 高 | 机会成本高 | 流行文化、短期项目、个体情绪 |
| **长期型** | 期限长，相对稳定 | 惯性大，变化缓慢 | 中 | 需考虑贴现 | 法律体系、语言、传统习俗 |
| **永恒型** | 追求超越个体期限 | 强烈传承机制，生命力持久 | 低 | 传承成本高 | 宗教、文明、数学定理 |

#### 7.2 期限来源

| 类型 | 定义 | 可调性 | 示例 |
| :--- | :--- | :--- | :--- |
| **自然期限** | 由 M1/M4 硬约束决定 | 基本不可调 | 人的寿命、资源耗竭、恒星生命周期 |
| **混合型期限** | 自然基础 + 人为干预 | 部分可调 | 经医疗延长的寿命、可续期的专利权 |
| **设定期限** | 完全由规则迭代决定 | 完全可调 | 合同期限、政策有效期、公司战略周期 |

#### 7.3 传承能力

| 类型 | 特征 | 机制 | 示例 |
| :--- | :--- | :--- | :--- |
| **可传承型** | 能超越个体期限 | 规则固化、信念传递、载体更新、意义框架延续 | 文化、制度、宗教、家族企业 |
| **一次性型** | 随载体消亡而终结 | 无传承机制 | 个体行为、临时任务、一次性工具 |


## 2.1.3 动力结构层：功能分化、活动模态与意义风格

### 一、概述

动力结构层是系统分类学的第三层，回答一个核心问题：**系统内部如何组织以处理可能性？其动力倾向和价值风格是什么？**

如果说 L1 基底约束层提供了导向结构的“物质前提”，L2 台阶与边界层划定了导向结构的“有效覆盖范围”，那么 L3 动力结构层刻画的就是导向结构的**内部配置**——系统在这个范围内**如何行动、如何选择、如何赋予意义**。

**本层包含七个维度**：
1. **功能分化类型**：四核心如何配置？
2. **主导活动模态**：系统对可能性空间的基本倾向是什么？
3. **导向结构可塑度**（新增）：导向结构能被主动重构的程度？
4. **意义场枢纽的价值风格**：意义枢纽如何回答“什么值得”？（六维框架）
5. **意义导向核心的路径风格**：系统如何运作其价值自由？（守护型/探索型/整合型）
6. **上层信息层内部结构**：信息处理系统如何内部组织？
7. **调用模式**：四核心权重的实际分布（可量化）。

### 二、与哲学核心的对接（v11.4）

| 哲学核心概念 | 在本层中的体现 |
| :--- | :--- |
| **导向结构** | 本层刻画导向结构的**内部配置**——功能分化、活动模态、调用模式是其直接表现。 |
| **四核心** | 功能分化类型直接对应四核心的权重配置；调用模式是可量化的权重分布。 |
| **存在活动四阶段** | 主导活动模态（试探/维持/防御）对应实现阶段的不同策略取向。 |
| **意义场与意义导向核心** | 区分“意义场枢纽”（所有系统都有）与“意义导向核心”（仅高阶系统）。 |
| **意义场的连续谱** | **导向结构可塑度**（新增维度）直接对应——原始型（原始意义场）、学习型（学习意义场）、反思型（反思意义场）。 |
| **痕迹的动态导向** | 调用模式是历史痕迹在统计上的固化——系统的“风格”就是其导向结构的宏观表现。 |
| **努斯能力** | 意义导向核心的路径风格（尤其是整合型）依赖努斯能力的五维支撑。 |
| **三层模型** | 上层信息层内部结构是本层的子维度，直接对应第十四章。 |

### 三、维度一：功能分化类型

功能分化类型刻画系统**四核心的宏观配置**——哪个核心在系统中占据主导地位。

#### 3.1 五种基本类型

| 类型 | 主导核心 | 应对的稀缺性 | 行为特征 | 典型系统 |
| :--- | :--- | :--- | :--- | :--- |
| **信用主导型** | 信用核心 | 资源稀缺 + 存续稀缺 | 重视储备、稳定、跨期配置 | 银行、保险机构、资源型企业 |
| **主动性主导型** | 主动性核心 | 能力稀缺 | 重视探索、创新、快速响应 | 初创企业、科研团队、探险组织 |
| **话语权主导型** | 话语权核心 | 协同稀缺 | 重视规则制定、秩序维护、权力分配 | 政府、立法机构、行业协会 |
| **意义导向核心主导型** | 意义导向核心 | 存续稀缺（整合） | 重视价值指向、文化传承、意义赋予 | 宗教组织、教育机构、文化团体 |
| **均衡型** | 四核心协同 | 综合应对 | 稳健而灵活，适应性强 | 成熟文明、健康个体、长寿企业 |

#### 3.2 功能分化类型的存在层级分布

| 层级 | 常见功能类型 | 说明 |
| :--- | :--- | :--- |
| **自然层** | 无明确功能分化 | 自然存在缺乏功能分化 |
| **生命层** | 主动性主导、信用主导 | 动物多主动性主导，植物多信用主导 |
| **社会层** | 话语权主导、意义导向核心主导、均衡型 | 政府多话语权主导，宗教多意义导向核心主导 |
| **技术层** | 主动性主导、信用主导 | 创新技术多主动性主导，成熟技术多信用主导 |
| **个体层** | 全类型 | 个体差异大 |
| **行为层** | 情境决定 | 行为的功能倾向由具体情境触发 |

### 四、维度二：主导活动模态

主导活动模态刻画系统对可能性空间的**基本行为倾向**。

#### 4.1 三种活动模态

| 活动模态 | 定义 | 对可能性空间的作用 | 风险特征 | 主导范式倾向 |
| :--- | :--- | :--- | :--- | :--- |
| **试探性主导** | 面向未知，试图扩展可能性边界 | 向外拓展，推动跃迁 | 高风险、高不确定 | 发展范式 |
| **维持性主导** | 面向已知，维持既有可能性空间 | 稳定边界，积累储备 | 低风险、可预期 | 中性/范式切换期 |
| **防御性主导** | 面向威胁，防止可能性空间收缩 | 向内巩固，防止退行 | 低风险、紧急响应 | 危机范式 |

#### 4.2 活动模态与演化阶段的关联

| 演化阶段 | 典型主导模态 | 说明 |
| :--- | :--- | :--- |
| **萌芽期** | 试探性主导 | 大量探索，寻找可行模式 |
| **成长期** | 试探性 + 维持性 | 探索与积累并行，快速扩张 |
| **成熟期** | 维持性主导 | 高效运作已有模式，探索减少 |
| **僵化期** | 维持性 + 防御性 | 固守旧模式，对威胁敏感但无力创新 |
| **衰退期** | 防御性主导 | 全力防止崩溃，主动性萎缩 |
| **转型期** | 试探性重新激活 | 探索新可能，重构动力结构 |

### 五、维度三：导向结构可塑度（新增）

导向结构可塑度刻画系统导向结构能够被主动重构的程度。这一维度直接对应哲学核心第九章中**意义场的连续谱**，是连接分类学与动力学的关键桥梁——它决定了系统的范式切换成本、努斯能力需求和演化灵活性。

| 类型 | 定义 | 意义场状态 | 导向结构特征 | 范式切换成本 | 典型系统 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **原始型** | 导向结构完全由物理定律或本能固化，无法主动调整 | 原始意义场 | 导向完全固化，只能按既定规则响应 | 几乎无穷大（无法主动切换） | 粒子、简单物理系统、低级生命 |
| **学习型** | 导向结构可通过试错和强化学习缓慢调整，但枢纽节点变更困难 | 学习意义场 | 导向权重可更新，但结构惯性大 | 高（需反复试错，路径依赖强） | 动物、传统社会、成熟期组织 |
| **反思型** | 导向结构可被元认知主动审视和重新配置 | 反思意义场 | 枢纽-边缘结构可主动重构 | 相对较低（可通过认知2和努斯能力主动调整） | 人类个体、成熟文明、高努斯组织 |

**与努斯能力的关联**：
- 原始型：努斯能力几乎为零。
- 学习型：初步具备试错学习能力，但元信念整合和反思性学习较弱。
- 反思型：具备完整的努斯五维能力，可主动重构导向结构。

**诊断意义**：导向结构可塑度决定了系统在环境剧变时的适应潜力。高可塑度系统能够以较低成本完成范式切换；低可塑度系统则极易陷入分化陷阱。

### 六、维度四：意义场枢纽的价值风格（六维框架）

意义场枢纽是意义场中高度固化的导向结构——所有系统都有。其价值风格由六个必然维度刻画，这些维度从四核心协同的需求中必然衍生。

#### 6.1 六维框架总览

| 必然维度 | 回应哪个核心的需求 | 核心问题 | 类型学（示例） |
| :--- | :--- | :--- | :--- |
| **价值优先级标准** | 信用核心 | 什么值得投入资源？ | 效率优先/公平优先/荣耀优先/传承优先/救赎优先 |
| **时间叙事** | 主动性核心 | 机会是否稀缺？自我是否有能力把握？ | 焦虑型/宿命型/审慎型/从容型 |
| **正当性来源** | 话语权核心 | 规则的终极合法性来自哪里？ | 神圣意志/自然法/理性契约/传统习俗/民意/绩效 |
| **边界划分原则** | 三核心共同 | 谁属于“我们”？边界如何划定？ | 血缘/地缘/文化认同/契约/信念 |
| **个体-集体锚定** | 三核心共同 | 价值的最终承载者是个体还是集体？ | 个体优先/集体优先/动态平衡 |
| **反信念容纳度** | 三核心共同 | 如何处理被主流排斥的可能性和声音？ | 压制型/容忍型/对话型/整合型 |

#### 6.2 各维度详解

**价值优先级标准**：系统配置稀缺资源时的优先级排序。效率优先追求单位投入的最大产出；公平优先追求分配正义；荣耀优先追求声望与历史地位；传承优先追求代际延续；救赎优先追求超越性的终极价值。

**时间叙事**：系统如何看待机会的稀缺性与自我的能动性。焦虑型（机会稀缺+自我渺小）急切冒险但易恐慌；宿命型（机会充裕+自我渺小）消极等待；审慎型（机会稀缺+自我重要）精心选择、稳健保守；从容型（机会充裕+自我重要）大胆试探、从容迭代。

**正当性来源**：规则的终极合法性根基。神圣意志型不可质疑；自然法型可被理性审视；理性契约型来自自愿同意；传统习俗型基于历史延续；民意型随多数意愿波动；绩效型基于实际效果。

**边界划分原则**：“我们”的边界如何划定。血缘型包容性最低；地缘型需长期融入；文化认同型可通过教化转化；契约型按规则加入退出；信念型对信者极高包容，对不信者极低。

**个体-集体锚定**：价值的最终承载者。个体优先型创新活力高但凝聚力可能弱；集体优先型动员能力强但可能压制个体；动态平衡型兼有活力与凝聚，协调成本高。

**反信念容纳度**：如何处理被排斥的可能性。压制型短期稳定但长期僵化；容忍型有一定多样性但缺乏整合；对话型主动寻求理解；整合型吸收为更高层次合题的素材。

### 七、维度五：意义导向核心的路径风格

意义导向核心是**高阶系统**才具备的、主动管理意义场的功能模块。其路径风格刻画系统**如何运作其价值自由**。

| 路径风格 | 核心特征 | 记忆/创新/传承侧重 | 努斯能力需求 | 典型系统 |
| :--- | :--- | :--- | :--- | :--- |
| **守护型** | 维护既有意义场枢纽结构 | 记忆主导，传承次之，创新较弱 | 较低 | 传统文明、宗教组织、保守型制度 |
| **探索型** | 持续生成新的价值指向 | 创新主导，记忆次之，传承较弱 | 中等 | 初创企业、科研前沿、艺术先锋 |
| **整合型** | 在新旧价值冲突中寻找更高层次框架 | 记忆、创新、传承三者均衡 | 高 | 成熟文明、长寿企业、智者 |

### 八、维度六：上层信息层内部结构

上层信息层内部结构刻画系统**信息处理系统的内部组织方式**，适用于具有复杂信息处理能力的系统。

| 类型 | 特征 | 感知区 | 注意区 | 概念区 | 典型系统 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **未分化型** | 无明确分区 | — | — | — | 简单生命、早期AI |
| **感知-注意分化型** | 有感知区和注意区，概念区未形成 | 发达 | 发达 | 弱 | 高级动物、实时响应系统 |
| **三区分化型** | 三区完备，功能清晰 | 发达 | 发达 | 发达 | 人类、高级AI、成熟组织 |
| **注意主导型** | 注意区特别发达，概念区相对简单 | 中等 | 极发达 | 简单 | 实时交易系统、应急响应中心 |
| **概念主导型** | 概念区特别发达，注意区相对简单 | 中等 | 简单 | 极发达 | 知识库系统、学术机构 |

### 九、维度七：调用模式

调用模式是系统在事件中调用四核心的**实际权重分布**，可量化。

\[
\mathbf{w} = (w_C, w_A, w_D, w_M), \quad \sum w = 1
\]

| 指标 | 定义 | 意义 |
| :--- | :--- | :--- |
| **平均调用模式** | 近期事件中调用权重的平均值 | 系统的“风格” |
| **调用模式方差** | 调用模式在各事件间的波动程度 | 风格稳定性——方差小则可能僵化 |
| **情境适应性** | 不同情境下调用模式的差异度 | 灵活性——差异大则适应性强 |

**与导向结构的关系**：调用模式是意义场中导向权重分布的统计表现。当特定核心相关的枢纽节点权重普遍较高时，系统呈现该核心主导的调用模式。调用模式的变化，本质上是导向结构在统计层面的迁移。


## 2.1.4 演化位置层：生命周期、生态位与关系网络

### 一、概述

演化位置层是系统分类学的第四层，回答两个核心问题：**系统处于生命周期的哪个位置？在生态中扮演什么角色？**

如果说 L1-L3 刻画的是导向结构的“内在属性”，那么 L4 刻画的是导向结构的**历史相位**和**生态位角色**——它在时间中的演化相位、在空间和社会网络中的生态位、以及与其他系统的关系。

**本层包含四个维度**：
1. **演化阶段**：系统处于生命周期的哪个阶段？
2. **动态类型**：系统在生态中扮演什么角色？
3. **垂直关系**：系统与部分/整体如何关联？
4. **平行关系**：系统与同层存在如何关联？

### 二、与哲学核心的对接（v11.4）

| 哲学核心概念 | 在本层中的体现 |
| :--- | :--- |
| **导向结构** | 本层刻画导向结构的**历史相位**——演化阶段反映导向结构的固化/可塑程度，动态类型反映其生态位角色。 |
| **演化** | 演化阶段直接对应系统在生命周期中的位置，是导向结构在时间中展开的宏观相位。 |
| **分化陷阱** | 僵化期的核心风险——导向结构过度专门化后，相互耦合过紧导致的整体适应性丧失。 |
| **量级淘汰** | 动态类型的转化（变异体→前驱→主流→残存）受量级淘汰驱动。 |
| **四核心** | 演化各阶段的四核心状态不同；垂直关系中的代理关系对应 M3 代理必然。 |
| **三层模型** | 垂直关系中的组成、涌现、控制、支持关系，在三层模型中有不同体现。 |

### 三、维度一：演化阶段

演化阶段刻画系统在**生命周期**中的位置。它不是独立的维度，而是对前三层（L1-L3）状态的综合判断。

#### 3.1 六阶段特征与判断依据

| 阶段 | L1 基底约束 | L2 台阶与边界 | L3 动力结构 | 可能性空间 |
| :--- | :--- | :--- | :--- | :--- |
| **萌芽期** | 信用极少，资源高度稀缺 | 台阶低，认知框架初成 | 功能未分化；试探性主导 | 极小但开放 |
| **成长期** | 信用快速增长 | 台阶跃升中，边界快速外移 | 功能分化形成；试探性+维持性 | 快速拓展 |
| **成熟期** | 信用充裕，成本均衡 | 台阶稳定，认知框架成熟 | 功能完善；维持性主导 | 最大化利用 |
| **僵化期** | 信用错配，成本刚性上升 | 台阶停滞，认知框架固化 | 枢纽固化；维持性+防御性 | 开始收缩 |
| **衰退期** | 信用枯竭，成本失控 | 台阶退化，认知框架失效 | 功能瓦解；防御性主导 | 急剧收缩 |
| **转型/消亡期** | 信用重构或归零 | 台阶切换或崩溃 | 动力结构重组或瓦解 | 重构后重新开放或消失 |

#### 3.2 演化阶段的动力学判据

| 判据 | 萌芽期 | 成长期 | 成熟期 | 僵化期 | 衰退期 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **存在力变化率** | 低但为正 | 快速上升 | 峰值，趋近于零 | 开始缓慢下降 | 持续为负 |
| **效率系数 ρ** | 不稳定，波动大 | 上升趋势 | 高且稳定 | 开始下降 | 持续低于1.0 |
| **试探频率** | 高（相对信用） | 高 | 中等 | 低 | 趋近于零 |
| **反信念容纳度** | 高 | 中高 | 中等 | 极低 | 低 |

#### 3.3 分化陷阱在演化阶段中的位置

**分化陷阱**是僵化期的核心风险——系统因在四个维度上过度专门化，相互耦合过紧，导致整体适应性丧失。诊断指标包括：
- 技术更新周期远长于环境变化周期
- 枢纽信念长期不变，边缘声音被压制
- 协同成本占系统总成本比例持续上升
- 效率系数 ρ 持续下降但无结构调整

### 四、维度二：动态类型

动态类型刻画系统在**生态系统**中的相对位置——它与主流存在的关系。

| 类型 | 与主流的关系 | 可能性空间 | 典型演化阶段 | 典型系统 |
| :--- | :--- | :--- | :--- | :--- |
| **主流存在** | 自身占据主导生态位 | 最大，但可能僵化 | 成熟期、僵化期 | 行业龙头、执政党、主流文化 |
| **前驱存在** | 边缘，蕴含未来主导特征 | 小但开放，增长潜力大 | 萌芽期、成长期 | 颠覆性技术、新兴思潮、初创企业 |
| **残存存在** | 曾主导但已衰退 | 急剧收缩 | 衰退期 | 传统技艺、旧制度残余、濒危语言 |
| **变异体** | 在主流内部产生的局部突变 | 极小，依附于主流 | 萌芽期前 | 内部创新项目、基因突变、新想法 |

**转化路径**：变异体 → 前驱 → 主流 → 残存 → 消亡（可能局部复兴）。

### 五、维度三：垂直关系

垂直关系刻画系统在**嵌套结构**中与部分和整体的关联。

| 关系类型 | 方向 | 定义 | 典型示例 |
| :--- | :--- | :--- | :--- |
| **组成关系** | 部分→整体 | 下级存在是上级存在的组成部分 | 器官与人体、部门与公司 |
| **涌现关系** | 部分→整体（质变） | 上级存在从下级耦合中产生新性质 | 意识从神经中涌现、文化从个体互动中涌现 |
| **控制关系** | 整体→部分 | 上级约束下级的主动性 | 政府与企业、大脑与肢体 |
| **支持关系** | 整体→部分 | 上级为下级提供资源或秩序 | 国家与公民、平台与商户 |
| **代理关系** | 双向 | 上级代表下级行使某些功能 | 议会与选民、CEO与股东 |

### 六、维度四：平行关系

平行关系刻画系统在**同一层级**中与其他存在的横向关联。

| 关系类型 | 对各方存在力的影响 | 典型示例 |
| :--- | :--- | :--- |
| **竞争关系** | 零和或负和 | 企业竞争、物种竞争、意识形态竞争 |
| **合作关系** | 正和 | 供应链合作、科研合作、国际联盟 |
| **寄生关系** | 负和（寄生者获益，宿主受损） | 寄生虫、专利侵权、网络攻击 |
| **共生关系** | 正和 | 根瘤菌与豆科植物、平台与优质商户 |
| **捕食关系** | 零和（捕食者获益，猎物消亡） | 掠食者与猎物、大企业吞并小企业 |


## 2.1.5 分类矩阵与应用：系统定位卡与典型案例

### 一、概述

本文件是系统分类学的综合与升华。它将 L1-L4 四个层次的全部维度整合为一个统一的**分类矩阵**，为任意存在提供系统化的“身份画像”方法。

分类矩阵是连接分类学与动力学的桥梁：分类学提供系统的**当前快照**（导向结构的静态配置），动力学研究系统的**演化轨迹**（导向结构的动态迁移）。

### 四、典型系统画像示例

#### 4.1 成熟期稳态企业

- **L1**：物质+信息混合，层级存在，资源稀缺度低，协同稀缺度中，物质成本主导
- **L2**：工业-信息台阶之间，认知框架实证+经验，社会层+技术层
- **L3**：信用主导或均衡型，维持性主导，**导向结构可塑度：学习型**，价值优先级效率优先，时间叙事审慎型，路径风格守护型或整合型
- **L4**：成熟期，主流存在，垂直关系以控制+支持为主，平行关系以竞争+合作为主
- **核心张力**：维持效率与防止僵化的平衡

#### 4.2 科学共同体

- **L1**：信息存在为主，层级存在，能力稀缺度中，信息成本主导
- **L2**：信息台阶，认知框架实证思维（部分系统思维），社会层+技术层
- **L3**：主动性主导，试探性主导，**导向结构可塑度：反思型**，真理获取方式实证型，时间叙事从容型，反信念对话型，路径风格探索型或整合型
- **L4**：成熟期（但保持更新），主流或前驱，垂直关系以涌现+代理为主，平行关系以合作+竞争为主
- **核心张力**：范式稳定性与革命性突破的张力

#### 4.3 传统宗教组织

- **L1**：信息存在为主，层级存在，资源稀缺度低，协同稀缺度中
- **L2**：农业-工业台阶，认知框架权威-启示型，社会层
- **L3**：意义导向核心主导，维持性主导，**导向结构可塑度：学习型（偏向原始）**，正当性来源神圣意志，边界划分信念型，个体-集体锚定集体优先，反信念压制型或容忍型，路径风格守护型
- **L4**：成熟期或僵化期，主流或残存，垂直关系以控制+传承为主
- **核心张力**：经典权威与现代价值的冲突

### 五、与系统动力学的接口（v12.0）

| 分类学提供的输入 | 动力学如何使用 | 涉及动力学章节 |

| L4 演化阶段 | 确定系统在范式切换周期中的位置，预警分化陷阱 | 累积与演化 |
| L4 动态类型 | 判断生态位压力和竞争态势 | 时机政治、危机传导 |
