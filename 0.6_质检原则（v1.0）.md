## 0.6_质检原则（v1.0）

---

### 一、概述

质检原则是 PPyQC 体系对自身进行质量评估的元标准，为各模块和整体版本的质检提供统一的维度定义、权重分配和评分细则。它源于认知2（反思性认知）对体系建构过程的自觉审视，与哲学核心中的“结构型建构合法性标准”（S1 内部自洽、S2 现象对接、S3 不可或缺）共同构成体系自我辩护与持续优化的方法论基础。

本文件定义的八维质检原则（QC8）是在 v10.7 版本基础上对原有七维标准（QC7）的优化，新增了“演化性”维度，调整了权重分配，并细化了认知效率的评分方法。质检原则与 [0.4_质检报告总览](./0.4_质检报告总览.md) 配套使用：质检报告记录评分结果，质检原则提供评分依据。

---

### 二、质检维度总览

| 维度       | 缩写  | 定义                   | 权重   |
| -------- | --- | -------------------- | ---- |
| **严谨性**  | R   | 概念边界清晰，定义无歧义，术语使用一致  | 0.15 |
| **逻辑性**  | L   | 体系内部自洽，推理无矛盾，因果关系成立  | 0.15 |
| **合理性**  | Rs  | 与科学共识相容，可被经验验证或证伪    | 0.10 |
| **辩证性**  | D   | 体现对立统一，把握复杂关系的动态平衡   | 0.10 |
| **全面性**  | C   | 覆盖核心现象领域，无重大遗漏       | 0.10 |
| **精准性**  | P   | 术语统一，模块接口清晰，层级划分合理   | 0.10 |
| **认知效率** | CE  | 信息表达高效，学习曲线平缓，认知负荷低  | 0.15 |
| **演化性**  | E   | 可扩展至新领域，适应未来变化，具备开放性 | 0.15 |

**权重合计**：1.00

---

### 三、各维度评分细则

每个维度采用 0–10 分制评分（可保留一位小数）。以下为各维度的评分参考标准。

#### 3.1 严谨性（R）

| 得分区间  | 标准                                  |
| ----- | ----------------------------------- |
| 9–10  | 所有核心概念均有严格定义，边界无模糊，无歧义术语；跨文档使用完全一致。 |
| 7–8.9 | 主要概念定义清晰，极少数边缘概念稍显模糊，但不影响整体理解。      |
| 5–6.9 | 部分概念定义含混，存在术语混用现象，需依赖上下文推测。         |
| <5    | 概念定义缺失或严重歧义，体系难以把握。                 |

#### 3.2 逻辑性（L）

| 得分区间  | 标准                                |
| ----- | --------------------------------- |
| 9–10  | 体系内部无任何矛盾，所有推理链条完整，因果关系明确，可形式化验证。 |
| 7–8.9 | 基本无矛盾，偶有推理跳跃但可补充，不影响核心结论。         |
| 5–6.9 | 存在局部矛盾或循环论证，削弱体系可信度。              |
| <5    | 重大逻辑矛盾，体系自洽性崩溃。                   |

#### 3.3 合理性（Rs）

| 得分区间  | 标准                                     |
| ----- | -------------------------------------- |
| 9–10  | 与相关学科（物理、生物、社会、心理等）科学共识高度一致，可设计实证检验方案。 |
| 7–8.9 | 基本符合科学共识，少数假设暂无法验证但无悖于已知事实。            |
| 5–6.9 | 部分观点与科学共识冲突或缺乏可验证性。                    |
| <5    | 严重违背科学常识或纯属思辨无法验证。                     |

#### 3.4 辩证性（D）

| 得分区间  | 标准                                     |
| ----- | -------------------------------------- |
| 9–10  | 深刻揭示对立统一关系（如稳定与变革、个体与整体），并能指导实践中的动态平衡。 |
| 7–8.9 | 体现常见辩证关系，但分析深度或应用指导略有不足。               |
| 5–6.9 | 存在辩证思维，但流于表面或二元对立。                     |
| <5    | 缺乏辩证性，处理复杂关系时简单化。                      |

#### 3.5 全面性（C）

| 得分区间  | 标准                                             |
| ----- | ---------------------------------------------- |
| 9–10  | 覆盖存在论、认识论、价值论及主要应用领域（自然、生命、社会、技术、个体、行为），无重大遗漏。 |
| 7–8.9 | 覆盖核心领域，但个别次领域未涉及或深度不足。                         |
| 5–6.9 | 主要领域有遗漏，解释力受限。                                 |
| <5    | 严重片面，无法解释关键现象。                                 |

#### 3.6 精准性（P）

| 得分区间  | 标准                            |
| ----- | ----------------------------- |
| 9–10  | 术语体系统一，模块间接口清晰，层级划分严格，无冗余或交叉。 |
| 7–8.9 | 基本清晰，偶有术语复用但可区分，接口较明确。        |
| 5–6.9 | 术语混用，模块边界模糊，需大量上下文说明。         |
| <5    | 术语混乱，无法独立使用。                  |

#### 3.7 认知效率（CE）

认知效率由**表达效率**（信息无损度）和**学习效率**（认知负荷）两个子维度构成，各自评分后再取几何平均。

**子维度一：表达效率（EE）**

| 得分区间  | 标准                                                    |
| ----- | ----------------------------------------------------- |
| 9–10  | 概念清晰度极高，无歧义；跨层转换（γ→β→α）信息保留率 >95%；符号系统语义互斥性（CE1）完全实现。 |
| 7–8.9 | 概念基本清晰，跨层转换信息保留率 80%–95%；CE1 基本实现。                    |
| 5–6.9 | 存在歧义概念，跨层转换信息损失明显；CE1 部分实现。                           |
| <5    | 概念含混，跨层转换失真严重；CE1 未实现。                                |

**子维度二：学习效率（LE）**

| 得分区间  | 标准                                       |
| ----- | ---------------------------------------- |
| 9–10  | 新手入门时间短（预估 < 8 小时），常见误解率极低；插件按需加载，核心负担轻。 |
| 7–8.9 | 入门时间适中（8–20 小时），偶有误解；插件机制有效。             |
| 5–6.9 | 入门时间长（20–40 小时），常见误解较多；插件机制不完善。          |
| <5    | 极难入门（>40 小时），误解普遍；无有效减负机制。               |

**认知效率最终得分**：
\[
CE = \sqrt{EE \times LE}
\]
（EE 和 LE 均采用 0–10 分制）

#### 3.8 演化性（E）

| 得分区间  | 标准                                             |
| ----- | ---------------------------------------------- |
| 9–10  | 体系设计明确考虑未来扩展，插件架构成熟，可轻松纳入新领域；核心概念抽象度适当，能覆盖新现象。 |
| 7–8.9 | 具备扩展性，但需一定调整；核心概念对新现象有一定覆盖能力。                  |
| 5–6.9 | 扩展需较大改动，核心概念对新现象解释力有限。                         |
| <5    | 体系封闭，无法适应新领域。                                  |

### 四、与结构型建构合法性标准（S1-S3）的对应

| 质检维度         | 对应 S1–S3                 |
| ------------ | ------------------------ |
| 严谨性、逻辑性      | S1 内部自洽                  |
| 合理性、全面性、演化性  | S2 现象对接                  |
| 辩证性、精准性、认知效率 | S3 不可或缺（体现概念的必要性和体系的精炼性） |

质检原则是 S1–S3 在质量评估层面的具体展开，使合法性标准转化为可操作的度量工具。

---

### 五、评分计算方法

1. **各维度评分**：由评估者根据上述细则独立打分（0–10 分）。
2. **综合得分**：
   \[
   \text{综合得分} = \sum_{i=1}^{8} (w_i \times s_i)
   \]
   其中 \(w_i\) 为权重，\(s_i\) 为维度得分。
3. **认知效率子项处理**：先分别评出 EE 和 LE，计算 CE = √(EE×LE)，再代入综合得分。
4. **结果呈现**：综合得分可保留三位小数，并附雷达图直观展示各维度表现（参见 [0.5_核心概念可视化模型](./0.5_核心概念可视化模型.md) 中的示例）。
